# Design Context

Design intent lives in **`../PRODUCT.md`** (strategy) and **`../DESIGN.md`** (visual system) at the repo's outer root — read them before UI work.

- **What it is:** *Libro Mayor de Albion* — a live arbitrage/flipping dashboard for Albion Online. Register **product**, platform **web**. UI in Spanish. For experienced traders; success is measured as craft (portfolio piece) at a Linear/Stripe bar.
- **North Star:** "The Merchant's Ledger" — a lamplit walnut desk; parchment pages, tear-off receipts, a terracotta wax seal on what matters.
- **Load-bearing principles:** the math is the product (surface the decision, not raw prices); game-native, never gamer-neon; playful in moments, precise across the whole; respect the expert (dense, minimal hand-holding); meaning never rests on color alone (buy/sell + risk need a label/shape beside the color).
- **Never:** raw-spreadsheet data dumps · gamer-neon/RGB dark UI · gray/blue/pure-black-or-white · tracked all-caps eyebrows.
